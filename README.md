## Usage
1. Clone the app:
 `git clone https://github.com/mhuskey/todo_lists.git`
2. Install gem dependencies:
`bundle install`
3. Migrate the databases:
`rails db:migrate`
4. Start the rails server:
`rails server`
5. Open your browser to the local address:
`http://0.0.0.0:3000/`
